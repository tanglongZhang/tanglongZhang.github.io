1.总述

2019年，总的来说平平顺顺，没有太大的波折，有收获也有烦恼。有时候会觉得很累，我不觉得这是坏事，反而觉得是好事，因为付出和回报总是成正比的，累了，但是也成长了。

2.简单总结2019

从2月份开始至今，关于今年所做事项，不做详细描述，自己体会。

  • 参与项目12个，验收项目12个

  • 挂名项目15个

  • 组建5个项目团队

  • 参与项目总天数250+天

  • 提交需求变更30+个，其中通过25个

  • 总培训人次270+/人次

  • 总计修改项目管理计划50+版

  • 验收材料修改40+版

  • 学会使用Hexo + Github +Coding搭建个人博客

  • 熟练应用禅道进行项目管理

  • 看完关于项目管理书籍10+

  • 喝酒喝到吐20+次

  • 买了车

  • 组织排练年会小品节目

总的来说，做的事情不多，但是也算是有点收获吧。

3.展望2020

2020是一个很有希望的一年，通过了2019的沉淀，我的能力有了一定的提升，
